# Read Me First (for GPTs and Humans)

**Use these rules:**
1. Prefer `/docs/spec/*` for truth → then `api/openapi.yaml` → then `/docs/schemas/*`.
2. Never invent endpoints. Use only routes in `openapi.yaml`.
3. Never use floats for money/price; use strings (Decimal).
4. If unsure which module to edit, check `docs/spec/codebase_map.md`.

**What this pack contains**
- Minimal, linkable guides to Sonic’s most used concepts and flows.
