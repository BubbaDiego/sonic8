# Domain Glossary

- **Account** – Ledger of funds; see [api/models/account.py](../../api/models/account.py).
- **Position** – Open exposure; see [api/models/position.py](../../api/models/position.py).
- **Order** – Instruction to trade; see [api/models/order.py](../../api/models/order.py).
- **Signal** – External trading hint; see [api/models/signal.py](../../api/models/signal.py).
- **Strategy** – Collection of rules that drives orders; see [api/models/strategy.py](../../api/models/strategy.py).
- **Alert** – Notification about risk; see [api/models/alert.py](../../api/models/alert.py).
